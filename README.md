# 🎮 Catch The Ball Game using OpenCV

A fun computer vision mini project built using **Python** and **OpenCV**. The game challenges players to catch falling balls by moving a character left and right. It demonstrates image processing, object animation, collision detection, and real-time rendering using OpenCV.

---

## 📌 Project Overview

This project is a simple 2D game developed with OpenCV that combines computer vision techniques with interactive gameplay. The player controls a character using the keyboard to catch falling balls while avoiding missed catches. The score increases for every successful catch, and the player has limited lives.

This project is ideal for beginners who want to understand how OpenCV can be used beyond image processing to create interactive applications.

## 🧠 Working Principle

1. The background image is loaded.
2. The player character is placed on the screen.
3. A ball falls from a random horizontal position.
4. The player moves left or right to catch the ball.
5. If the ball lands within the catching area:
   - Score increases.
6. Otherwise:
   - One life is lost.
7. The game continues until all lives are exhausted.
